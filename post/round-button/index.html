Creating A Round Button With React Native Vector Icons

May 25, 2017

I had a design I wanted to replicate and the button in this design is a round button and it has an icon in it.

Here is a snapshot

I immediately set out to accomplish the task. In my head, it was going to take less than 10 minutes. Ooh was I wrong. It took me the whole day to figure out.

I’ll try to make the process be less painful for you and for my future self (I tend to forget alot.)

Let’s get started

Make sure you have your react-native project up and running. What you want to do next is install react-native-vector-icons in your project like so

  $ npm install react-native-vector-icons --save
  $ react-native link

If the react-native link doesn’t work for you, please check the guides to do this manually.

After the installation and linking phase, you should quit the app and run react-native run-ios or react-native run-android again to build the icons with your app else, you would see a beautiful red screen staring at you.

We have the icons installed in our app, now its time to make use of them

Dont forget to import the Icon like so import Icon from 'react-native-vector-icons/Ionicons';. I am using Ionicons, you can check the list of the various supported icon fonts and use them as you wish.

Now let’s create the button

To accomplish this, I tried wrapping the <Icon /> in a View and styling that but that didnt work out. So I used the <Icon.Button /> instead. It’s a lot more flexible (I figured this out by looking through the source code, its JS afterall).

Here is the snippet of code to produce a round button with an icon in the middle

<Icon.Button 
  name="ios-remove" 
  size={25} 
  color='#fff' 
  backgroundColor='#fff' 
  style={{ borderRadius: 15, backgroundColor: '#bbb', height: 30, width: 30 }} 
  iconStyle={{ marginRight: 0 }}
/>

The name prop specifies what icon to show. For me, I went over to the Ionicons doc to get the specific icon I wanted.

The size prop specifies how big or how small you want the icon to be.

The color prop is just the color of the icon.

The backgroundColor is by default blue – #007AFF. To eliminate that, I set the backgroundColor to white. Feel free to play with this if white doesn’t work in your case.

Here, we provide the <Icon.Button /> component with a style props. This style props defines how the body of the Icon would be rendered. This component returns a TouchableHighlight and View so its the same styling properties you use for a View component you can supply to the style prop.

iconStyle has a default marginRight: 5. I don’t know why that makes sense to be there but what do I know. It’s very easy to overwrite which is what I did with the iconStyle props.

Hopefully, this helps you save a day unlike me.
